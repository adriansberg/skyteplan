## Spacing Scale

Declared values (multiples of 4):

| Token | px Value | Tailwind | Usage |
|-------|----------|----------|-------|
| xs | 4px | `p-1` / `gap-1` | Icon gaps, inline badge padding |
| sm | 8px | `p-2` / `gap-2` | Compact element spacing, badge px |
| md | 16px | `p-4` / `gap-4` | Default element spacing, card padding |
| lg | 24px | `p-6` / `gap-6` | Section padding |
| xl | 32px | `p-8` / `gap-8` | Major section breaks |
| 2xl | 48px | `py-12` | Page-level vertical rhythm |
| 3xl | 64px | `py-16` | Reserved (not used this phase) |

**Exceptions:**
- Top bar height: `h-10` (40px) — intentionally not a standard content spacing token
- Bottom tab bar height: `h-14` (56px) + `env(safe-area-inset-bottom)` via inline style
- Touch targets on tabs: minimum 44px tall — achieved by tab height 56px (exceeds minimum)
- Page content bottom padding: `pb-16` (64px) — ensures content clears fixed tab bar
- Sticky date header offset: `scroll-mt-10` (40px) — matches slim top bar height

## Typography

| Role | Size | Tailwind Size | Weight | Tailwind Weight | Line Height | Usage |
|------|------|---------------|--------|-----------------|-------------|-------|
| Body / data | 16px | `text-base` | 600 | `font-semibold` | 1.5 | All data text: shooter names, event names, scores |
| Label / meta | 12px | `text-xs` | 600 | `font-semibold` | 1.4 | Badge labels, metadata (Skive, Lag, class), timestamps |
| Heading | 20px | `text-xl` | 700 | `font-bold` | 1.25 | Date section headers, shooter name in detail cards |
| Page title | 24px | `text-2xl` | 700 | `font-bold` | 1.2 | Page-level `<h1>` on each route |

**Monospace:** Times and scores use `font-mono` at their declared size. Applied to:
- `checkinDateTime` formatted output (time strings)
- Series `sum` and `sumInner` score values
- Shot value displays (`shot.valueInt.shot.valueDec`)

**Weights in use: 2** — `font-semibold` (600) for all non-heading text, `font-bold` (700) for headings. Size contrast (12px vs 16px) provides sufficient visual hierarchy between label/meta and body/data roles.

## Layout Contract

### Top Bar (`+layout.svelte`)

```
<header class="sticky top-0 z-40 h-10 border-b border-neutral-200 bg-neutral-50 flex items-center justify-between px-4">
  <a href="/"><img src={stordalenLogo} alt="Stordalen Skytterlag" class="h-8 w-auto" /></a>
  <RefreshButton />
</header>
```

- Height: `h-10` (40px)
- No site name text visible (logo image only)
- No nav links (removed entirely — navigation moves to BottomTabBar)
- `z-40` — below bottom tab bar `z-50`

> Source: CONTEXT.md D-01, D-02

### Bottom Tab Bar (`BottomTabBar.svelte` — new file)

```
<nav
  class="fixed bottom-0 left-0 right-0 z-50 flex h-14 border-t border-neutral-200 bg-neutral-50"
  style="padding-bottom: env(safe-area-inset-bottom)"
>
  <!-- Three tabs: Skyteplan / Skyttere / Premieliste -->
</nav>
```

**Tab anatomy (per tab):**

| State | Classes |
|-------|---------|
| Active | `flex-1 flex flex-col items-center justify-center border-t-2 border-emerald-600 text-emerald-600 text-xs font-medium` |
| Inactive | `flex-1 flex flex-col items-center justify-center border-t-2 border-transparent text-neutral-500 text-xs font-medium` |

- Label text: "Skyteplan", "Skyttere", "Premieliste" — plain text, no emojis, no SVG icons
- `env(safe-area-inset-bottom)` applied as inline `style=` (not Tailwind — not a utility)
- Bottom tab bar border-top: `border-t border-neutral-200` (Claude's discretion pick — D-Context)
- Active detection: reuse `$derived(page.url.pathname)` pattern from current layout

> Source: CONTEXT.md D-03, D-04, D-06; REQUIREMENTS.md UX-01

### Page Content Wrapper (all three routes)

All outermost page containers must add `pb-16` to prevent bottom tab bar overlap:

```svelte
<div class="container mx-auto px-4 py-4 pb-16">
```

> Source: CONTEXT.md D-04

### Sticky Date Headers (`+page.svelte`)

```svelte
<div
  use:registerTodaySection={isToday}
  class="sticky top-10 z-30 scroll-mt-10 bg-neutral-50 border-b border-neutral-200 px-3 py-3 sm:px-6 sm:py-4"
>
  <h2 class="text-xl font-bold text-neutral-900">
    {dateLabel}
    <span class="ml-1 text-xs font-normal text-neutral-600">({events.length} skyttere)</span>
  </h2>
</div>
```

- `sticky top-10` — offset matches top bar `h-10` (40px)
- `scroll-mt-10` — replaces `scroll-mt-14 sm:scroll-mt-20`
- No gradient, no shadow on sticky header
- "I dag" label produced by existing `getDateLabel()` — no logic change needed
- `z-30` — below top bar `z-40` and bottom tab bar `z-50`

> Source: CONTEXT.md D-14, D-15, D-16; REQUIREMENTS.md UX-04

## Interaction Contract

| Interaction | Behavior |
|-------------|----------|
| Tab tap (active) | No action — already on route |
| Tab tap (inactive) | SvelteKit `<a href>` navigation — standard |
| Tab active state | Driven by `$derived(page.url.pathname)` — instant, no transition |
| Pull-to-refresh | Existing `PullToRefresh.svelte` — no changes |
| Sticky header scroll | Native CSS sticky — no JS needed |
| Auto-scroll to today | Existing `registerTodaySection` action + `scrollIntoView` — `scroll-mt-10` offset only change |
| `<details>` expand | Native HTML — no JS; existing arrow rotate animation retained |
| Badge pulse | `animate-pulse` on `ongoing` status only |
